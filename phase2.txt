2. Sort tasks by:

   - Tight SLA window

3. For each task t:
    bestSlot = -1
    bestPenalty = ∞

    For slot s in [l_t, u_t]:
        If conflict exists → skip
        If resource exceeds → skip
        Temporarily assign t → s
        Compute penalty
        If better:
            update bestSlot

    If no slot found:
        Try LOCAL REPAIR
        If fails → return infeasible

    Assign permanently

4. Return assignment

Try:
    Remove conflicting task
    Reassign both tasks
If success → accept
